Astro は偶数バージョンのnodeしかサポートしていないので 24系を使う。
mise が入ってるので使っていく。

```
mise install node@24
mise use node@24
node -v
24.14.1
```

```
npm create astro@latest
```

## Cloudflare にデプロイする
https://docs.astro.build/ja/guides/deploy/cloudflare/

アダプタをインストール
```
npx astro add cloudflare
```
